# 大金台球赛事管理系统

一个专为中式台球线下赛事设计的网页应用，支持从报名到赛程完结的全流程管理。

## 🎱 功能特性

### 1. 报名系统
- 支持多种赛事类型：9球追分、6球追分、4球追分、中式八球标准赛/让球赛/弹性让球赛，可自定义
- 公开扫码报名，生成独立报名页面二维码
- 收集选手姓名、电话、参赛项目、种子排名备注
- 展示报名费金额，上传微信/支付宝收款码
- 后台审核报名（通过/拒绝），支持手动添加/删除选手

### 2. 对阵表生成与管理
- 支持单败淘汰赛、双败淘汰赛
- 自动生成对阵表，支持轮空自动判定晋级
- 手机端全屏查看，点击对阵卡片录入实时比分
- 更新比分后所有查看页面自动同步
- 生成赛事流程图，清晰展示夺冠路径

### 3. 实时统计与数据预览
- 首页展示：剩余选手数、已完赛场次、待完赛场次
- 选手数据统计：胜场数、负场数、胜率
- 弹性让球赛支持自动/手动更新让球数
- 分享赛事总页面和对阵表，他人可直接查看实时数据

### 4. 品牌展示
- 大金台球连锁店品牌信息
- 三家分店：韩家墅店、青光店、王庆坨店
- 联系电话：4000-000-357
- 科技感绿色主题，潮流设计

## 🚀 快速开始

### 本地运行
1. 将项目文件夹复制到任意位置
2. 双击 `index.html` 用浏览器打开
3. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8080
   
   # Node.js
   npx serve .
   ```
4. 浏览器访问 `http://localhost:8080`

### 部署说明
- 纯前端应用，无需后端服务器
- 数据存储在浏览器 LocalStorage 中
- 建议在同一台设备上管理赛事，或导出数据备份

## 📱 使用流程

### 创建赛事
1. 进入「后台」→「创建赛事」
2. 填写赛事名称、类型、规则、报名费
3. 选择参赛门店（可多选）
4. 上传微信/支付宝收款码图片
5. 点击「创建赛事」

### 选手报名
1. 在「赛事管理」中点击「显示报名二维码」
2. 选手扫码进入报名页面
3. 填写信息并选择参赛门店
4. 扫码支付报名费，保留截图
5. 提交报名等待审核

### 生成对阵表
1. 审核通过所有参赛选手
2. 点击「自动生成对阵表」
3. 系统随机分配对阵位置
4. 点击「查看对阵表」或「复制对阵链接」分享给选手

### 录入比分
1. 在对阵表页面点击任意比赛
2. 输入双方比分
3. 保存后系统自动推进晋级
4. 所有查看页面实时同步更新

## ⚙️ 系统设置

### 配置访问地址
在「后台」→「系统设置」中填写网站访问地址，用于生成正确的二维码链接。

示例：
- 本地：`http://192.168.1.100:8080`
- 公网：`https://your-domain.com/billiards`

### 数据备份
- 导出数据：生成 JSON 文件备份
- 导入数据：恢复之前导出的数据

## 🎨 技术栈

- HTML5 + CSS3 + JavaScript（原生，无框架依赖）
- LocalStorage 本地数据存储
- QRCode.js 二维码生成

## 📝 注意事项

1. **数据存储**：所有数据保存在浏览器本地，清除浏览器数据会丢失，请定期导出备份
2. **多设备同步**：如需多设备管理，建议导出数据文件手动同步
3. **收款码**：请在创建赛事时上传清晰的收款码图片，方便选手支付
4. **网络要求**：生成二维码需要配置正确的访问地址，确保选手能正常访问

## 🔧 自定义配置

### 修改门店信息

```html

  --green-dark: #00cc6a;   /* 深色 */
  --bg-dark: #050f0a;      /* 背景色 */
  /* ... */
}
```
